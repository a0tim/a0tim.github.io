OpenCV環境設置教學

 OpenCV(for Java in Eclipse)環境介面安裝教學

STEP1    下載三個opencv套件

●  javacv-0.4-bin.zip

●  javacv-0.4-cppjars.zip

●  OpenCV-2.4.4.exe

STEP2    先點選OpenCV-2.4.4.exe執行

解壓縮路徑建議設為"C:\"根目錄，點選"Extract"開始解壓縮

STEP3    解壓縮時間攸關硬碟I/O速度及CPU效能
解壓完畢約2.7GB，請確認硬碟空間足夠

STEP4    接著解壓縮另外兩個檔案，建議將解壓路徑設為與STEP2相同路徑

STEP5    接著開啟STEP2解壓完後的"opencv"資料夾，
進入"\build"，接著依照系統位元選取"\x86"或"\x64"，
再進入"\mingw\bin"，複製路徑文字

STEP6    XP版本回到桌面，點選"我的電腦"右鍵，點選"內容"。
W7版本回到桌面，點選"電腦"右鍵 ，點選"內容"。

STEP7    點選"進階系統設定">"環境變數"

STEP8    在下方的[系統變數]欄找到"Path"變數並點選"編輯"，
最後面加上分號";"後將剛才複製的路徑貼上

STEP9    開啟Eclipse，新增一個Java Project

先命名，JRE部分點選第三個(圖中圈選處)，點選"Configure JRE's"

選取"jre7"後點選"Edit..."

STEP10    點選"Add External JARs"，將框框內5個jar檔案加入，
必須注意綠色部分請依照自己系統選擇

加入完畢後點選"Finish"完成，點選"OK"結束

再點選"Finish"結束

STEP11    最後確認5個jar檔是否都成功匯入到Project內

STEP12    完成，你可以開始撰寫OpenCV-Java程式了
